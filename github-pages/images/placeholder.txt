Esta carpeta es para las imágenes de tu sitio web.

Imágenes recomendadas:
- hero-fashion.jpg (1200x600px) - Imagen principal del hero
- producto1.jpg, producto2.jpg, etc. (400x400px) - Imágenes de productos
- categoria-vestidos.jpg (600x400px) - Imagen de categoría vestidos
- categoria-chaquetas.jpg (600x400px) - Imagen de categoría chaquetas
- categoria-accesorios.jpg (600x400px) - Imagen de categoría accesorios
- destacado1.jpg, destacado2.jpg, destacado3.jpg (400x400px) - Productos destacados

Consejos:
- Formato: JPG o PNG
- Tamaño máximo: 1MB por imagen
- Optimiza las imágenes antes de subirlas
- Usa nombres descriptivos sin espacios ni caracteres especiales

Para agregar imágenes:
1. Sube los archivos a esta carpeta en GitHub
2. Actualiza las rutas en index.html
3. Los cambios aparecerán automáticamente en tu sitio
